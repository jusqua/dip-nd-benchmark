# Digital Image Processing N-Dimensional Benchmark

## Requirements

- Intel LLVM Compiler
- AdaptiveCpp Compiler
- CUDA Compiler
- MATLAB
- [this specific version of the VisionGL](https://github.com/jusqua/visiongl/archive/83bb5314a4ff9ff487fdc6328caebf7505f3db66.tar.gz)
- libtiff

## Usage

```bash
./run.sh [ROUNDS]
```
